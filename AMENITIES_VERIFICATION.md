## Acceso al Sistema

### Para Administradores:
1. Ir a: `http://localhost:3000/admin`
2. Login: `admin` / `roccosa226`
3. Buscar pestaña **"Comodidades"** en el menú lateral
4. Seleccionar propiedad del dropdown
5. Agregar/editar comodidades por categoría

### Datos por Defecto:
Cada propiedad ya tiene comodidades predefinidas en el contexto admin que pueden ser editadas.

## Troubleshooting

Si no ves la pestaña "Comodidades":

1. **Refrescar página completamente**: Ctrl+F5 / Cmd+Shift+R
2. **Limpiar caché del navegador**
3. **Verificar consola del navegador** por errores
4. **Reiniciar servidor**: Ctrl+C y `npm run dev`
